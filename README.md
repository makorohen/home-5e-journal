# home-5e-journal

本仓库用于归档和发布一系列独立的 HTML 页面。这些页面各自独立，不依赖于特定的域名或在线服务，适合作为静态资源存档或离线浏览使用。

## 项目简介

home-5e-journal 是一个以归档为目的的仓库，存放着多个自行编写的 HTML 文件。每个文件均为自包含的网页，可能包含样式、脚本及结构化的文本内容。仓库不针对任何特定网站或平台，仅作为页面资料的组织和存储。

## 目录说明

仓库根目录下直接放置 HTML 文件，部分文件可能包含同名的资源文件夹。目录结构如下：

```
home-5e-journal/
├── index.html
├── page-01.html
├── page-02.html
├── assets/
│   └── (样式、图片、脚本等资源)
└── ...
```

- 每个 HTML 文件对应一个独立的页面。
- `assets/` 目录存放页面共用的资源文件（如有需要）。
- 建议通过本地文件系统或任何静态服务器直接打开 HTML 文件浏览。

## 页面归档说明

- 所有页面均为纯前端实现，无需后端支持。
- 页面内容以中文为主，部分页面可能包含英文或混合语言。
- 归档内容不承诺定期更新，但会视情况添加或修正。
- 页面不追踪用户行为，不含分析脚本或外部数据请求（除非明确标注）。
